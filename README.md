# 📓 OrganizaAí

## 1. Introdução

Atualmente muitas pessoas possuem dificuldade para organizar suas tarefas e estudos devido à rotina corrida. Pensando nisso, foi desenvolvido o protótipo de aplicação web chamado **OrganizaAí**, que tem como objetivo auxiliar na organização diária, acompanhamento de metas e gerenciamento de estudos.

O sistema foi projetado para ser simples, intuitivo e acessível diretamente pelo navegador, sem necessidade de instalação de banco de dados ou servidores complexos.

---

## 2. Objetivo do Projeto

O principal objetivo do OrganizaAí é ajudar estudantes e profissionais a manterem suas atividades organizadas através de ferramentas simples de produtividade.

Além disso, o sistema utiliza conceitos de gamificação para incentivar o usuário a concluir tarefas e atingir metas, tornando a experiência mais motivadora.

---

## 3. Funcionalidades Implementadas

### 3.1 Cadastro de Tarefas

O usuário pode adicionar tarefas diárias e visualizar todas elas em uma lista organizada.

### 3.2 Conclusão de Tarefas

As tarefas podem ser marcadas como concluídas, permitindo acompanhar o progresso diário.

### 3.3 Sistema de Metas

O sistema possui um espaço para definição de metas, ajudando no acompanhamento dos objetivos pessoais ou acadêmicos.

### 3.4 Calendário Simples

Foi implementado um calendário básico para auxiliar na organização das atividades e compromissos.

### 3.5 Salvamento Automático

Todas as informações são armazenadas utilizando o recurso **localStorage** do navegador, permitindo que os dados permaneçam salvos mesmo após fechar a aplicação.

### 3.6 Modo Escuro

O usuário pode alternar entre tema claro e escuro para melhorar a experiência visual durante o uso.

### 3.7 Gamificação

Ao concluir tarefas, o usuário acumula pontos, incentivando a produtividade e o cumprimento das atividades planejadas.

## 5. Estrutura do Sistema

O sistema foi dividido em três partes principais:

### Front-end

Responsável pela interface gráfica e interação com o usuário.

### Lógica de Funcionamento

Desenvolvida em JavaScript para controlar tarefas, metas, pontuação e armazenamento.

### Armazenamento Local

Utilização do localStorage para salvar os dados diretamente no navegador.

---

## 6. Exemplo de Funcionamento

1. O usuário acessa o sistema.
2. Adiciona suas tarefas do dia.
3. Define uma meta de estudos.
4. Marca as tarefas conforme forem concluídas.
5. Recebe pontos pelas atividades realizadas.
6. Os dados permanecem salvos automaticamente para acessos futuros.

---

## 7. Conclusão

O OrganizaAí é uma aplicação simples voltada para organização pessoal e produtividade. O projeto demonstra a utilização de tecnologias web modernas para resolver problemas do dia a dia de estudantes e profissionais.

Além de facilitar o gerenciamento de tarefas e estudos, o sistema utiliza gamificação para aumentar o engajamento dos usuários, tornando a organização mais prática e motivadora.
